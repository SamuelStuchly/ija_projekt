--- Project IJA 2020 - Public transport simulator --- 

Authors:
    - Samuel Stuchlý (xstuch06)
    - Samuel Spišák (xspisa02)

This project is an aplication that simulates movement of public transport vehicles on the street layout.
It allows to display of vehicles on the map layout according to current time, or custom time set by user.
User can scale the passage of time and simule heavier traffic conditions on chosen streets.
